# 停止运动

## 任务信息

任务值 (TID): **37**

任务描述：

- 机器人停止运动，进入高阻尼状态。此任务通常用于紧急停止或在需要立即停止所有运动时使用。
- 需要配合 [执行器使能](/fourier-grx-N1/docs/tasks/servo_on) 任务使用，确保电机处于使能状态。
